# Astro - Sitio web para abogado

Sitio web premium para un despacho de abogados con estilo luxury navy & gold.

## Cómo usar

1. Instalar dependencias con `pnpm install`
2. Iniciar el servidor de desarrollo con `pnpm dev`

## Estructura

- `src/pages/index.astro` — página principal
- `public/styles/global.css` — paleta de colores y estilo global

> Recuerda usar `pnpm` como gestor de paquetes.
